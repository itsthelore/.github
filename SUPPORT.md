# Support

Thanks for using Lore. Here's where to go for help:

- **Documentation** — <https://itsthelore.github.io/rac-core/> covers the
  quickstart, CLI reference, and MCP server setup. Wayfinder and Proofkeeper
  document themselves in their own READMEs.
- **Questions and ideas** — open a
  [discussion or issue on rac-core](https://github.com/itsthelore/rac-core/issues)
  for anything about the engine, the corpus model, or the MCP server; use the
  specific repository's issues for tool-specific questions.
- **Bugs** — open an issue on the affected repository using its bug template.
- **Security issues** — never in a public issue; see [SECURITY.md](SECURITY.md).

Everything here is early and evolving quickly, so questions about intended
behaviour are as welcome as bug reports.
